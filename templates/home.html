About this system

Customers page:
You can see all the customers, add a new customer, search customer by name and delete a customer.
Books page:
You can see all the books, search book by name, add a new book, loan a book and delete a book. When you want to loan a new book, as a Librarian you need first to find the customer who want to loan this book and to see his id. Then you can put his id in the loan's form.
Loans:
You can see all loans, search loan by customer name or by book name, see late returns, return a loan. In this table you can identify the loan's status by color.
